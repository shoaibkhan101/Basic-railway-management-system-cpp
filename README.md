# 🚆 Railway Management System (C++)

A console-based Railway Management System built in C++, using Programming Fundamental concepts such as classes and objects and arrays.

## 📌 Features
- Manage train details (add, update, view)
- Book and cancel tickets
- Display train schedules
- Search train information

## 🛠️ Built With
- **Language:** C++
- **Concepts:** (Classes & Objects)

## 🚀 How to Run
```bash
g++ railway_management.cpp -o railway_management
./railway_management
```

## 📖 About
This project was developed as part of coursework at Air University Islamabad, focused on applying C++ principles to a real-world system design problem.
